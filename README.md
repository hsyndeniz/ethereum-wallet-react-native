## :hammer: Build it yourself

Run the following commands:

```bash

# Clone this repo
git clone https://github.com/hsynran/ethereum-wallet-react-native.git && cd ethereum-wallet-react-native

# Install dependencies
npm install

# Run the app on ios
npm run ios

# Run the app on android
npm run android
```
